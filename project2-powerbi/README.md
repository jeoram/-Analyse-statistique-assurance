# Project 2 — Power BI Insurance Dashboard

## Objective

This project is designed to show a strong Data Analyst profile for an insurance and reporting role, especially aligned with a position like the one offered by Covéa.

It emphasizes:
- Power BI dashboard design
- KPI clarity and business relevance
- data documentation and maintainability
- reporting logic and user-oriented design

## Business case

The objective is to create a reporting solution for insurance business users who need a clear view of:
- customer portfolio performance
- claims and premium trends
- satisfaction and churn signals
- operational service indicators

## What is included

- a business-oriented dataset
- a Power BI-ready CSV export
- a documentation file for KPI logic and data sources
- a dashboard structure proposal

## Structure

- data/ : dataset used for reporting
- outputs/ : CSV exports and documentation
- powerbi/ : Power BI-related assets

## Deliverables

- dashboard-ready data export
- KPI documentation
- reporting design recommendations

## Tools used

- Python
- pandas
- Power BI
- Excel / CSV for data handoff

## How to run

```bash
python main.py
```

## Key reporting topics

- premium evolution
- claims frequency
- customer satisfaction
- churn indicators
- service quality metrics
